# blog

## Cloner le projet

`git clone https://github.com/LeBonBateau/blog.git`

## Pousser des modifications
### Publier sur prod
`./cmd/deploy.sh`

### Pousser les mise à jour sur Github
`git add .`
`git commit -m "Ton message"`
`git pull`
`git push `

/!\ Mot de passe nécessaire lors du `deploy` et du `push`.
